HOW TO USE
──────────
1. Extract this ZIP file to any folder on your computer.
2. Open "index.html" in any modern web browser
   (Chrome, Firefox, Edge, Safari all work).
3. Create your profile on first launch.
4. Start tracking!

FEATURES
────────
✅ Dashboard          - Overview of today's spending & monthly status
✅ Expense Tracker    - Daily expense logging with 10 categories
✅ Income Manager     - Monthly income recording
✅ Savings & Credit   - Month-end processing & accumulation tracking
✅ Lend List          - Track money lent to others
✅ Reports            - Charts, category breakdowns, 6-month trends
✅ CSV Export         - Download monthly reports
✅ Settings           - Profile management & data reset

MONTH-END WORKFLOW
──────────────────
1. Enter your monthly income at the start of the month
2. Log daily expenses throughout the month
3. At month end → Go to "Savings & Credit" → Click "Close Month"
4. System auto-calculates:
   - If Income > Expenses → Amount added to Savings ✅
   - If Expenses > Income → Deficit added to Credit ⚠️ (with alert)

DATA STORAGE
────────────
All data is stored in your browser's localStorage.
Data stays on YOUR device — nothing sent to any server.
Clearing browser data will erase tracker data.
Use "Export CSV" in Reports to backup your data.

TECHNICAL
─────────
- Pure HTML/CSS/JavaScript (no server needed)
- Works offline after first load
- Chart.js for visualizations (loaded from CDN)
- Responsive — works on mobile and desktop
